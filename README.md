# 💊 Pharma Kanban 4.0

## 📌 Description
Pharma Kanban 4.0 is a web application developed to manage and track pharmaceutical production using the Kanban method.

This project was built using:
- HTML
- CSS
- JavaScript
- GitHub Pages (Deployment)

---

## 🚀 Features

✅ Kanban Board (À Faire / En cours / Contrôle Qualité / Terminé)  
✅ Drag & Drop system  
✅ QR Code generation for each production lot  
✅ Production statistics dashboard  
✅ Add new production lot (dynamic form)  
✅ Team management page  
✅ Data saved using LocalStorage  
✅ Fully responsive design
